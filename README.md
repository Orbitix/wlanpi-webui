# wlanpi-webui

This is a local WebUI customized for the WLAN Pi.

# script deployment

```
cd ~
wget https://raw.githubusercontent.com/joshschmelzle/wlanpi-webui/main/setup.sh
sudo chmod +x setup.sh
sudo ./setup.sh
```

Validate by browsing to `http://<WLANPI_IP>`.
