profiler is a tool that helps determine the capabilities of wireless clients. this is accomplished by automating analysis of association frames.

  • instructions

    to activate the profiler:

    • ensure a supported WLAN NIC is plugged into the WLAN Pi
    • start the profiler process using the Front Panel Menu System (FPMS)
    • Menu > Apps > Profile > Start

    to profile a client:

    • once started the configured SSID will broadcast (default: "WLAN Pi")
    • connect a client and enter any random 8 characters for the PSK
    • the client will expectedly fail authentication but we should receive the association request

    once a client is profiled refresh this page to see new results.
